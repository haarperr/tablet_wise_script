Comandos por texto

Abaixo estão alguns comandos que você pode usar pelo chat dentro do jogo

Comandos usuais:

/roupa
Ex: /roupa mascara 40
Funciona para trocar as peças de roupa pelo número ID dela (funciona para jaqueta, camisas, máscaras, capacete, etc) Como sei o id da peça de roupa?

Comandos para comunicação:

/forarp Fora do RP (Falar fora do personagem ou reportar algo de anti-rp)
/twt Bate-Papo (Dentro do RP)
/ilegal Totalmente anônimo para realizar venda de armas, comprar informação de drogas, etc (Dentro do RP)
/olx Anunciar vendas (Dentro do RP)
/911 Escrever denúncia para a Polícia (Dentro do RP)
/121 Pedir por socorro para o DHHS (Dentro do RP)

Comandos por teclas (hotkeys)

Abaixo estão alguns comandos que você pode usar pelo teclado dentro do jogo

Comandos de animações:

[F1] Faz com que seu personagem fique em posição de guarda
[F2] Faz com que seu personagem coloque a mão na cabeça
[F6] Faz com que seu personagem mostre o dedo do meio
[F10] Faz com que seu personagem cruze os braços
[X] Faz com que seu personagem levante as mãos
[Z] Faz com que seu personagem desmaie

Comandos usuais:

[SHIFT] Modo Drift
[Num9] Desativa/Ativa o Modo Drift (já vem ativo)
[L] Abre o celular
[K] Abre o menu de jogador
